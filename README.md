# Juego Secreto JK

## Descripción

Juego Secreto JK es un juego de adivinanza de números desarrollado en JavaScript. El objetivo del juego es adivinar un número secreto generado aleatoriamente entre 1 y 10. Es un proyecto simple y educativo que demuestra conceptos básicos de JavaScript, manipulación del DOM y lógica de juegos.

## Funcionalidad

- **Generación de número secreto**: El juego genera un número aleatorio entre 1 y 10 al inicio de cada partida.
- **Adivinanza**: El usuario ingresa un número en el campo de entrada y hace clic en "Intentar".
- **Pistas**: Si el número ingresado no es correcto, el juego proporciona pistas indicando si el número secreto es mayor o menor.
- **Contador de intentos**: El juego cuenta el número de intentos realizados por el usuario.
- **Reinicio del juego**: Una vez que el usuario adivina el número, puede iniciar un nuevo juego con un número secreto diferente.
- **Prevención de repeticiones**: El juego evita repetir números ya sorteados en rondas anteriores, aunque en este caso, se reinicia completamente.

## Cómo ejecutar

1. Clona o descarga el repositorio.
2. Abre el archivo `index.html` en un navegador web.
3. ¡Disfruta del juego!

## Tecnologías utilizadas

- **HTML**: Estructura de la página web.
- **CSS**: Estilos para la interfaz de usuario (archivo `style.css`).
- **JavaScript**: Lógica del juego (archivo `app.js`).

## Estructura del proyecto

- `index.html`: Archivo principal HTML.
- `style.css`: Hoja de estilos CSS.
- `app.js`: Código JavaScript del juego.
- `img/`: Carpeta con imágenes utilizadas en la interfaz.
